PROJECT OVERVIEW
================================================================================
This is a multi-functional Command Line Interface (CLI) application that combines
multiple core programming concepts including loops, functions, file handling, and
problem-solving. The application serves as a final mini project demonstrating
comprehensive Python skills.

================================================================================
FEATURES
================================================================================

CORE FEATURES (5 Required Features Implemented)
--------------------------------------------------------------------------------

1. CALCULATOR
   - Addition
   - Subtraction
   - Multiplication
   - Division
   - Input validation and error handling for division by zero

2. PATTERN GENERATOR
   - Star Pattern (Triangle shape)
   - Number Pattern (Pyramid shape)
   - User-defined number of rows

3. FILE HANDLING SYSTEM
   - Add data to file (appdata.txt)
   - View all stored data
   - Delete all data
   - Persistent storage using text files

4. NUMBER UTILITIES
   - Check Even or Odd numbers
   - Check Prime numbers
   - Generate Multiplication Table (1 to 10)

5. SIMPLE QUIZ SYSTEM
   - 5 multiple choice questions
   - Score calculation at the end
   - Score history saved to file (quizscores.txt)
   - View previous quiz scores

BONUS FEATURES (Optional Enhancements)
--------------------------------------------------------------------------------

6. LOGIN SYSTEM
   - Username and password authentication
   - Default credentials: admin / password123
   - 3 attempts before exit
   - Optional login (can be skipped)

7. COLORFUL CLI OUTPUT
   - ANSI color codes for better visual experience
   - Green for success and correct answers
   - Red for errors and warnings
   - Yellow for important information
   - Cyan for headers and borders

8. PASSWORD GENERATOR
   - Generate random strong passwords
   - Customizable password length (minimum 6 characters)
   - Includes letters, numbers, and symbols

9. NOTES APPLICATION
   - Write new notes with timestamps
   - Read all saved notes
   - Clear all notes
   - Persistent storage (mynotes.txt)

10. QUIZ SCORE STORAGE
    - All quiz scores saved to file
    - Timestamp for each quiz attempt
    - View complete score history

================================================================================
PROGRAM STRUCTURE
================================================================================

The application is organized into modular functions:

├── Utility Functions
│   ├── clearScreen() - Clear terminal display
│   ├── printHeader() - Display formatted headers
│   ├── printError() - Display error messages
│   ├── printSuccess() - Display success messages
│   ├── waitForEnter() - Pause execution
│   └── Input validation functions

├── Core Features
│   ├── calculatorMenu() - Calculator operations
│   ├── patternMenu() - Pattern generation
│   ├── fileHandlingMenu() - File operations
│   ├── numberUtilitiesMenu() - Number operations
│   └── quizMenu() - Quiz system

├── Bonus Features
│   ├── loginSystem() - User authentication
│   ├── generatePassword() - Password generation
│   ├── notesMenu() - Notes management
│   └── additionalUtilities() - Bonus features menu

└── Main Application
    ├── displayMainMenu() - Main menu display
    └── main() - Application entry point

================================================================================
INSTALLATION & REQUIREMENTS
================================================================================

PREREQUISITES
- Python 3.x installed on your system
- No external libraries required (uses only Python standard library)

INSTALLATION STEPS
1. Save the Python file as "cli_application.py"
2. Open terminal or command prompt
3. Navigate to the file directory
4. Run the application using: python cli_application.py

================================================================================
HOW TO USE
================================================================================

STARTING THE APPLICATION
1. Run the program: python cli_application.py
2. Choose whether to use login system (yes/no)
3. If yes, enter credentials: admin / password123
4. Main menu will appear with 7 options

MAIN MENU OPTIONS
1. Calculator - Perform arithmetic operations
2. Pattern Generator - Create star and number patterns
3. File Handling System - Manage data in files
4. Number Utilities - Analyze numbers and generate tables
5. Simple Quiz System - Take quizzes and view scores
6. Additional Utilities - Access password generator and notes
7. Exit Application - Close the program

NAVIGATION TIPS
- Enter the number corresponding to your choice
- Press Enter to confirm selections
- Use "Back to Main Menu" options to return
- Press Enter after viewing results to continue

================================================================================
FILE STRUCTURE
================================================================================

The application creates and manages the following files:

appdata.txt          - Stores user data from File Handling System
quizscores.txt       - Stores quiz scores with timestamps
mynotes.txt          - Stores notes from Notes Application

All files are created automatically in the same directory as the Python script.

ERROR HANDLING

TROUBLESHOOTING
================================================================================

ISSUE: Colors not showing in output
SOLUTION: Ensure terminal supports ANSI colors (most modern terminals do)

ISSUE: Files not being created
SOLUTION: Check write permissions in the current directory

ISSUE: Login fails
SOLUTION: Use default credentials: username "admin", password "password123"

ISSUE: Quiz scores not saving
SOLUTION: Verify write permissions for quizscores.txt file
